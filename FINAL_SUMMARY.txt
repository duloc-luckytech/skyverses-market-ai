FILE:     hooks/useProductImageEditor.ts

LINES:    576-615 (crop) and 704-742 (draw)
SEVERITY: 🔴 CRITICAL

ISSUE:
  When uploading local file + cropping:
  ✅ Upload succeeds
  ✅ Crop job created
  ❌ Polling for result FAILS SILENTLY
  ❌ UI hangs until 3-minute timeout

ROOT CAUSE:
  Line: const st = statusRes.data?.status;
  
  Problem: No validation that statusRes.data exists before accessing it
  - If API returns error or malformed response, .data is undefined
  - Optional chaining (?.  ) silently converts undefined to falsy
  - Code falls to else branch and loops until timeout
  - User sees spinner stuck for 3 minutes
