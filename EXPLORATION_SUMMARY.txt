2. TYPES & INTERFACES
================================================================================

MAIN PROJECT TYPE:
  ├─ PaperclipProject
  │  ├─ _id, name, budgetLimit
  │  ├─ activeModel: 'claude-sonnet' | 'claude-opus'
  │  ├─ agentConfigs: AgentConfig[]
  │  ├─ conversationThreads: Thread[]
  │  └─ taskHistory: ITaskResult[]

AGENT CONFIGURATION:
  ├─ AgentConfig
  │  ├─ deptId: string
  │  ├─ brief: string
  │  └─ enabledSkills: string[]

TASK RESULT:
  ├─ ITaskResult
  │  ├─ id, dept, prompt, output
  │  ├─ status: 'done' | 'error'
  │  └─ timestamp

CONVERSATION THREAD:
  ├─ Thread
  │  ├─ deptId: string
  │  └─ messages: { role, content }[]

5. AI MODELS & CONFIGURATION
================================================================================

AVAILABLE MODELS:
  1. Claude Sonnet 4 (claude-sonnet-4-6)
     → Fast, cheap, balanced
     → Recommended for most tasks
  
  2. Claude Opus 4 (claude-opus-4)
     → Most powerful
     → For complex reasoning/analysis

API LAYER:
  → All calls via backend proxy (secured)
  → aiChatStreamViaProxy() for streaming
  → aiChatOnceViaProxy() for single calls
  → buildSystemMessage() for prompt construction
  → parseAIJSON<T>() for structured responses

7. API ENDPOINTS
================================================================================

PAPERCLIP PROJECTS API (/paperclip-projects):
  GET    /                              List all projects
  GET    /:id                           Get full project
  POST   /                              Create new project
  PATCH  /:id                           Update metadata
  DELETE /:id                           Delete project
  PUT    /:id/agents                    Save agent configs
  PUT    /:id/threads                   Save threads
  POST   /:id/history                   Append task result

AUTHENTICATION:
  → Via AuthContext (hooks/context/AuthContext.tsx)
  → Uses getHeaders() for API Authorization
